## 注意事项

- TypeScript 7.0.2 未采用：typescript-eslint 8.65.0 的 TypeScript peer 上限仍为 `<6.1.0`。
- `@testing-library/jest-dom` 7 与 `@eslint-react/eslint-plugin` 5.17.3 未采用：它们要求 Node >=22，不能满足两套 SPA 的 Node 20 支持基线。
- lint-staged 17.1.0 未采用：它要求 Node >=22.22.1，高于 SPA 与 SSR 模板声明的最低版本。
- code-inspector-plugin 2 未采用：仍会引入需要构建许可的依赖链，模板保留 1.x 兼容线。
- pnpm 11.15.1 未采用：它要求 Node >=22.13，无法覆盖 SPA 模板的 Node 20 基线。
- 本地 Node 24.1.0 不满足 SSR 模板声明的 `^24.11.0`，SSR 命令会显示 engine warning；所有最终门禁仍实际完成并通过。
- React SSR 与 Vue SSR lint 仍有既有 Tailwind 任意值简化建议；Nuxt 构建会输出插件耗时提示，均不阻塞模板运行。
- `cwa-docs` 构建仍提示主 JavaScript chunk 约 613 kB，属于既有站点拆包优化项，不阻塞本次文档发布。
