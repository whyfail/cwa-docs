## 注意事项

- TypeScript 7.0.2 未采用：typescript-eslint 8 当前 peer 范围为 `<6.1.0`。
- Pinia 4.0.2 与 `@pinia/nuxt` 1.0.0 未采用：Vue Router 5.1.0 当前 peer 仍要求 Pinia 3.0.4。
- code-inspector-plugin 2 未采用：它引入 `@code-inspector/core` 与 `node-pty` 构建脚本，会触发 pnpm 构建许可门禁；模板保留 1.x 兼容线。
- pnpm 11.13.1 未采用：其 engine 要求 Node >=22.13，与两套 SPA 的 Node 20 支持基线冲突。
- 本地 Node 24.1.0 不满足 SSR 模板声明的 `^24.11.0`，因此 SSR 命令会显示 engine warning；命令本身均完成并通过。
- Nuxt 构建仍有上游 module-preload sourcemap 与 VueUse PURE 注释位置警告，构建产物和 E2E 不受影响。
- SSR lint 会提示少量 Tailwind 任意值可简化建议，不阻塞 lint 或模板运行。
